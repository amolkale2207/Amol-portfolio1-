Html

HTML (HyperText Markup Language) is the standard language for creating web pages. It structures content using elements like headings, paragraphs, links, and multimedia, forming the backbone of every website.

Css

CSS (Cascading Style Sheets) is used to style and layout web pages, controlling colors, fonts, spacing, and responsiveness. It enhances HTML by making websites visually appealing and user-friendly.

MySql

MySQL is a widely used open-source relational database management system, known for its speed, reliability, and scalability in handling data.

Bootstrap

Bootstrap is a popular front-end framework for building responsive, mobile-first websites quickly, using pre-designed components and grid systems.

JavaScript

JavaScript is a dynamic, high-level programming language used to create interactive and dynamic content on websites, enhancing user experience with features like animations and real-time updates.

Jquery

jQuery is a fast, lightweight JavaScript library that simplifies DOM manipulation, event handling, and AJAX interactions with minimal code.

Java

Java is a versatile, object-oriented programming language widely used for building platform-independent applications, from web and mobile to enterprise solutions.
